5. Calculer un itinéraire → Vérifier directions

```

### ✅ Test Stripe (3 min)
```
1. Créer une demande client
2. Accepter en tant qu'artisan
3. Compléter la mission
4. Payer avec : 4242 4242 4242 4242
5. Vérifier dans https://dashboard.stripe.com/test/payments

✅ Si paiement réussi → Stripe OK
```

```
CLIENT :
