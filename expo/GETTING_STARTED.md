## 🔧 PROBLÈMES FRÉQUENTS

### ❌ Carte ne s'affiche pas
```bash
# Solution
# Vérifier que la clé est dans .env
GOOGLE_MAPS_API_KEY=AIzaSyDFbRWjaRlK-RFgkpo-XizRR7x2_O5y1Mg

# Redémarrer l'app
npx expo start --clear
```

### ❌ Paiement échoue
```bash
# Solution
# Utiliser une carte de test Stripe
Numéro : 4242 4242 4242 4242
Date : 12/34
CVC : 123

# Vérifier que le mode test est activé
USE_STRIPE_TEST_MODE=true
```

### ❌ "No base url found"
```bash
# Solution
# Vérifier .env
EXPO_PUBLIC_RORK_API_BASE_URL=https://toolkit.rork.com

# Redémarrer
npx expo start --clear
```
